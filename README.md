# Conway_Game_of_Life
The following program is an implementation of Conway_Game_of_life with specific shapes implemented. The code is meant to simulate cells in a 2d space which come alive with based on certain conditions being met.Specific shapes are inserted it creates a process of movement as cells live(turn on) and then die.
